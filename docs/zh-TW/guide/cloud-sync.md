# 雲同步

得益於 PouchDB 和 CouchDB 強大的同步能力，Vnite 的資料庫原生支援多端即時雲同步，並且附帶了一個官方伺服器和帳號系統，為每位使用者提供 300MB 的同步配額。

當然，你仍可以選擇自託管 CouchDB，開箱即用，請參考以下文件。

- https://docs.couchdb.org/en/stable/
