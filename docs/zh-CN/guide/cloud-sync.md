# 云同步

得益于 PouchDB 和 CouchDB 强大的同步能力，Vnite 的数据库原生支持多端实时云同步，并且附带了一个官方服务器和账号系统，为每位用户提供 300MB 的同步配额。

当然，你仍可以选择自托管 CouchDB，开箱即用，请参考以下文档。

- https://docs.couchdb.org/en/stable/
